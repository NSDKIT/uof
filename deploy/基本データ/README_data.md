# 基本データ フォルダ

このフォルダには、シミュレーションに必要な入力データ（.mat形式）が格納されています。

## 含まれているファイル

| ファイル名 | 説明 | 単位 | サイズ |
|---|---|---|---|
| `PV_base_2019.mat` | 2019年度の既設PV容量（月別） | MW | 小 |
| `PV_base_2018.mat` | 2018年度の既設PV容量（月別） | MW | 小 |
| `PR_2019.mat` | 2019年度のシステム出力係数（月別） | - | 小 |
| `PR_2018.mat` | 2018年度のシステム出力係数（月別） | - | 小 |
| `MSM_bai_2019.mat` | 2019年度のMSM倍数係数（月別） | - | 小 |
| `MSM_bai_2018.mat` | 2018年度のMSM倍数係数（月別） | - | 小 |
| `irr_fore_data.mat` | 予測日射量データ（全日・全モード） | W/m² | 中 |
| `D_30min.mat` | 30分値需要データ（全日分） | MW | 小 |

## 別途配置が必要なファイル（容量大のため未収録）

以下のファイルは容量が大きいため、このパッケージには含まれていません。
担当者から直接受け取り、このフォルダに配置してください。

| ファイル名 | 説明 | 目安サイズ |
|---|---|---|
| `D_1sec.mat` | 1秒値需要データ（全日分） | 数百MB |
| `irr_mea_data.mat` | 実測日射量データ（全日・全モード） | 数百MB |

## データの構造

### PV_base_YYYY.mat
- 変数名: `PV_base`
- サイズ: (月数+3) × 3 の行列
- 3列目が各月の既設PV容量 [MW]

### PR_YYYY.mat / MSM_bai_YYYY.mat
- 変数名: `PR` / `MSM_bai`
- サイズ: 1 × 12 のベクトル（月別）

### irr_fore_data.mat
- 変数名: `irr_fore_data`
- サイズ: (365日 × 24時間) × モード数 の行列
- 各列が異なるPVパネル設置モードに対応

### D_30min.mat / D_1sec.mat
- 変数名: `D_30min` / `D_1sec`
- サイズ: 日数 × 時間ステップ数 の行列
- 各行が1日分のデータ
